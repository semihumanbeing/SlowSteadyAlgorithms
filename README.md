강의를 보고 문제를 이해할때까지 풀어보는 형식의 스터디
1. 우선 강의를 본다.
2. 문제를 풀 때, 처음에는 푸는방법을 보고 따라 쳐보고, 같은문제를 다시 풀고, 안되면 답을 보고 푸는 방법을 다시 공부한다.
3. 일주일에 1번 자기가 푼 문제 1개를 어떻게 풀었는지 과정을 설명한다
